API Documentation
=================

Documentation for the Python API (application programming interface) provided by the **MetagenomeDB** toolkit.

MetagenomeDB can be used in your Python scripts to add, remove, annotate and relate sequences and collections. Powerful queries can be designed to retrieve subsets of those data based on their properties or the properties of their relationships.
