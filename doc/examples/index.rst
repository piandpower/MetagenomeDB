Examples
========

Here is a list of examples of use of the MetagenomeDB api (see :doc:`../api/index`) to perform various tasks. Code is provided so that you can replicate those examples with your own data.

:doc:`graph_of_collections`
	How to generate a graph (viewable with `Cytoscape <http://www.cytoscape.org/>`_) with collections and their sub- or super-collections.

:doc:`graph_of_reads_sharing_contigs`
	How to generate a graph (viewable with `Cytoscape <http://www.cytoscape.org/>`_) with contigs sharing reads.

:doc:`querying_blast_results`
	How to retrieve best hits of sequences from previously imported BLAST outputs.
